IOS app to manually OCR license plates from screenshots/photos and blacklist Bolt drivers.

For "auto-OCR", set up the red rectangle first, then share the image with the app     
or    
Open a photo in the app and manually select the OCR region using the red rectangle   
Run OCR on the selected area    
If the plate is not on the blacklist, a flyout appears:    
Tap to add (optional note)    
Swipe to dismiss the flyout
